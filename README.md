# Eraste App
**originated by : Malik H**

Web Aplikasi no style eraste yang fleksibel dan mudah di setting  : 
1. ubah settingan database : 
- di config/database
- jalankan perintah : 

> cp .env.example .env

- lalu ubah .env nya
2. **jalankan perintah:** 

> php artisan migrate

3. **run app dengan :**

> php artisan serve
